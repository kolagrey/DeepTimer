## How to use this repo

### Clone this repo
```sh

$ git clone https://github.com/kolagrey/DeepTimer

```

### Install dependencies using

```sh

$ cd DeepTimer

$ yarn install 

```

### Start the app using 

```sh
    
    $ yarn start

```


## Testing

```sh
    
    $ yarn test

```
